# ❤️ Heart Disease Prediction Project

## Overview
Machine Learning project to predict heart disease using patient health data.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Model Training
- Prediction
- Visualization
- Accuracy Comparison

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
processed.cleveland.data

## Workflow
1. Data Collection
2. Data Preprocessing
3. Model Training
4. Evaluation
5. Visualization

## Run

```bash
pip install -r requirements.txt
python heart.py
```
